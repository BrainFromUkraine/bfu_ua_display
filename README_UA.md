# BFU UA Display

**Професійна бібліотека для відображення українського тексту на дисплеях MicroPython**

[![Ліцензія: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![MicroPython](https://img.shields.io/badge/MicroPython-1.19+-blue.svg)](https://micropython.org/)
[![ESP32](https://img.shields.io/badge/Platform-ESP32-green.svg)](https://www.espressif.com/en/products/socs/esp32)

Легка та оптимізована бібліотека для відображення українського тексту на дисплеях, які зазвичай використовуються з ESP32 та MicroPython проєктами. Вирішує проблему відсутності підтримки українських символів у стандартних бібліотеках для дисплеїв.

## 🎯 Навіщо потрібна ця бібліотека?

### Проблема

Коли ви намагаєтеся вивести український текст на OLED або LCD дисплей за допомогою стандартних функцій MicroPython, ви отримуєте:

```python
# Стандартний спосіб - НЕ ПРАЦЮЄ з українською!
display.text("ПРИВІТ", 0, 0)  # ❌ Показує крякозябри або нічого
```

**Чому так?** Стандартні шрифти в MicroPython не містять українські літери (А, Б, В, Г, Ґ, Д, Е, Є, Ж, З, И, І, Ї, Й та інші).

### Рішення

```python
# З BFU UA Display - ПРАЦЮЄ!
from bfu_ua_display import ua_text

ua_text(display, "ПРИВІТ УКРАЇНО!", 0, 0)  # ✅ Показує правильно!
```

## 📦 Встановлення

### Рекомендований спосіб: Через mip

```python
import mip
mip.install("github:BrainFromUkraine/bfu_ua_display")
```

Це встановить бібліотеку в `/lib/bfu_ua_display/` на вашому ESP32.

### Перевірка встановлення

Перевірте, що бібліотека встановлена правильно:

```python
# Тестуємо імпорт
from bfu_ua_display import ua_text, ua_text_center, ua_text_scaled
print("✓ BFU UA Display імпортовано успішно!")

# Перевіряємо версію
import bfu_ua_display
print(f"Версія: {bfu_ua_display.__version__}")
```

### Альтернатива: Ручне встановлення

Якщо ви віддаєте перевагу ручному встановленню або виникають проблеми з mip:

1. Завантажте файли бібліотеки з GitHub:
   - Перейдіть на https://github.com/BrainFromUkraine/bfu_ua_display
   - Завантажте папку `bfu_ua_display`

2. Завантажте на ваш ESP32 за допомогою:
   - **Thonny IDE** (найпростіше для початківців):
     1. Відкрийте Thonny IDE
     2. Підключіть ESP32
     3. View → Files
     4. Створіть папку `lib` на ESP32, якщо її немає
     5. Перетягніть папку `bfu_ua_display` в папку `lib`
   
   - **ampy**: `ampy put bfu_ua_display /lib/bfu_ua_display`
   - **rshell**: `rsync bfu_ua_display /pyboard/lib/`
   - **mpremote**: `mpremote cp -r bfu_ua_display :lib/`

### Очищення (якщо були проблеми з встановленням)

Якщо ви раніше встановили бібліотеку неправильно і маєте вкладені папки, спочатку очистіть:

```python
import os

# Видаляємо неправильне встановлення
try:
    os.rmdir('/lib/bfu_ua_display')
except:
    pass

# Потім перевстановлюємо через mip
import mip
mip.install("github:BrainFromUkraine/bfu_ua_display")
```

### Вирішення проблем

**Проблема:** `ImportError: no module named 'bfu_ua_display'`

**Рішення:** 
1. Перевірте, що бібліотека знаходиться в `/lib/bfu_ua_display/` на вашому ESP32
2. Переконайтеся, що папка містить файл `__init__.py`
3. Спробуйте перезавантажити ESP32: `import machine; machine.reset()`

**Проблема:** Вкладені папки типу `bfu_ua_display/font5x7.py/font5x7.py`

**Рішення:** Це було викликано старим методом встановлення. Видаліть папку `bfu_ua_display` повністю і перевстановіть, використовуючи рекомендований метод mip вище.

**Проблема:** `ValueError: Unsupported Transfer-Encoding: chunked`

**Рішення:** Ця помилка рідко виникає з поточним форматом package.json. Якщо ви її зустрінете, використовуйте метод ручного встановлення.

```python
from ssd1306 import SSD1306_I2C
from bfu_ua_display import ua_text, ua_text_center

# Ініціалізація дисплея

ua_text(oled, "ПРИВІТ УКРАЇНО!", 0, 0)
```

```python
from bfu_ua_display import ua_text_scaled

oled.show()
```

## 📖 Опис функцій

### Основні функції

#### `ua_text(display, text, x, y, color=1, bg_color=0, clear_bg=False)`

Виводить текст у вказаній позиції з підтримкою українських символів.

**Параметри:**
- `display` - Об'єкт дисплея з методом `pixel()`
- `text` - Рядок для виводу (українська, англійська, цифри, символи)
- `x` - Координата X (лівий край)
- `y` - Координата Y (верхній край)
- `color` - Колір тексту (за замовчуванням: 1 - білий)
- `bg_color` - Колір фону (за замовчуванням: 0 - чорний)
- `clear_bg` - Очищати фон за текстом (за замовчуванням: False)

**Повертає:** Загальну ширину виведеного тексту в пікселях

**Приклад:**
```python
ua_text(oled, "Температура: 25°C", 0, 0)
```

---

#### `ua_text_center(display, text, y, color=1, bg_color=0, clear_bg=False, display_width=128)`

Виводить текст по центру дисплея горизонтально.

**Параметри:**
- `display` - Об'єкт дисплея
- `text` - Рядок для виводу
- `y` - Координата Y (верхній край)
- `color` - Колір тексту (за замовчуванням: 1)
- `bg_color` - Колір фону (за замовчуванням: 0)
- `clear_bg` - Очищати фон (за замовчуванням: False)
- `display_width` - Ширина дисплея в пікселях (за замовчуванням: 128)

**Повертає:** Координату X, де було виведено текст

**Приклад:**
```python
ua_text_center(oled, "УКРАЇНА", 28)
```

---

#### `ua_text_scaled(display, text, x, y, scale=2, color=1, bg_color=0, clear_bg=False)`

Виводить текст зі збільшенням (2x, 3x тощо).

**Параметри:**
- `display` - Об'єкт дисплея
- `text` - Рядок для виводу
- `x` - Координата X (лівий край)
- `y` - Координата Y (верхній край)
- `scale` - Коефіцієнт масштабування (1=звичайний, 2=подвійний тощо)
- `color` - Колір тексту (за замовчуванням: 1)
- `bg_color` - Колір фону (за замовчуванням: 0)
- `clear_bg` - Очищати фон (за замовчуванням: False)

**Повертає:** Загальну ширину виведеного тексту в пікселях

**Приклад:**
```python
ua_text_scaled(oled, "ПРИВІТ", 0, 0, scale=2)
```

### Додаткові функції

Бібліотека також містить корисні функції в модулі `bfu_ua_display.utils`:

- `measure_text(text)` - Отримати розміри тексту
- `wrap_text(text, max_width)` - Перенесення тексту на новий рядок
- `truncate_text(text, max_width)` - Обрізання тексту з трикрапкою
- `center_position(text, display_width, display_height)` - Розрахувати центральну позицію
- `supports_ukrainian(text)` - Перевірити, чи містить текст українські символи
- `validate_text(text)` - Перевірити підтримку символів

## 🖥️ Підтримувані дисплеї

Наразі протестовано та працює з:

- **SSD1306** - OLED 128x64, 128x32 (I2C/SPI)

Бібліотека розроблена для роботи з будь-яким дисплеєм, що підтримує:
- `pixel(x, y, color)` - Встановлення окремого пікселя
- `fill_rect(x, y, width, height, color)` - Заповнення прямокутника (опціонально, для оптимізації)
- `show()` - Оновлення дисплея (опціонально, для буферизованих дисплеїв)

## 🔌 Підключення дисплея

### SSD1306 OLED (I2C)

**Схема підключення:**

```
ESP32          SSD1306
-----          -------
GPIO 22  --->  SCL (годинник)
GPIO 21  --->  SDA (дані)
3.3V     --->  VCC (живлення)
GND      --->  GND (земля)
```

**Код ініціалізації:**

```python
from machine import I2C, Pin
from ssd1306 import SSD1306_I2C

# Створюємо об'єкт I2C
i2c = I2C(0, scl=Pin(22), sda=Pin(21), freq=400000)

# Створюємо об'єкт дисплея (128x64 пікселі)
oled = SSD1306_I2C(128, 64, i2c)

# Тепер можна використовувати дисплей!
```

## 💡 Приклади використання

### Приклад 1: Простий текст

```python
from machine import I2C, Pin
from ssd1306 import SSD1306_I2C
from bfu_ua_display import ua_text

# Ініціалізація
i2c = I2C(0, scl=Pin(22), sda=Pin(21))
oled = SSD1306_I2C(128, 64, i2c)

# Очищаємо дисплей
oled.fill(0)

# Виводимо текст
ua_text(oled, "ПРИВІТ!", 0, 0)
ua_text(oled, "Україна 2026", 0, 10)

# Оновлюємо дисплей
oled.show()
```

### Приклад 2: Центрований текст

```python

ua_text_center(oled, "УКРАЇНА", 10)
```

### Приклад 3: Датчик температури

```python
from bfu_ua_display import ua_text
import time

while True:
    # Читаємо температуру (приклад)
    temp = 25.5
    
    # Очищаємо дисплей
    oled.fill(0)
    
    # Виводимо дані
    ua_text(oled, "Температура:", 0, 0)
    ua_text(oled, f"{temp}°C", 0, 12)
    ua_text(oled, "Вологість:", 0, 28)
    ua_text(oled, "60%", 0, 40)
    
    # Оновлюємо дисплей
    oled.show()
    
    # Чекаємо 1 секунду
    time.sleep(1)
```

```python
from bfu_ua_display import ua_text
```

## ❓ Часті питання

### Чому текст не відображається?

1. Перевірте підключення дисплея (SCL, SDA, VCC, GND)
2. Переконайтеся, що викликали `oled.show()` після виводу тексту
3. Перевірте, чи правильно ініціалізований I2C

### Як змінити розмір тексту?

Використовуйте функцію `ua_text_scaled()`:

```python
ua_text_scaled(oled, "ВЕЛИКИЙ", 0, 0, scale=2)  # Подвійний розмір
ua_text_scaled(oled, "ДУЖЕ ВЕЛИКИЙ", 0, 20, scale=3)  # Потрійний розмір
```

### Чи працює з іншими дисплеями?

Так! Бібліотека працює з будь-яким дисплеєм, що має метод `pixel()`. Просто передайте об'єкт вашого дисплея у функції.

### Скільки пам'яті використовує бібліотека?

Приблизно 2-3 КБ, що менше 1% від доступної пам'яті ESP32. Це дуже мало і безпечно для використання.
